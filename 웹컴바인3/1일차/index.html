문장, 표현식
자바스크립트는 문장과 표현식으로 이루어져 있다.

변수 var / let / const
정수형, 실수형, 문자, 문자열

1. 문자(a -> z, A -> Z), 밑줄(_)혹은 달러 기호($)로 시작해야 합니다. 두번째 글자부터는 숫자(0~9)도 가능
2. '대문자'와 '소문자'는 구별합니다. myname과 myName은 다른 이름이다.
3. 예약어는 사용하면 안된다. ex) if, for, var X
1. 의미 없는 이름은 좋지 않다.
2. 너무 추상적인 이름은 좋지 않다.
ex) let name;
3. 모든 변수 이름은 camelCase로 쓰는 것이 좋다.
